# Instruksi Penggunaan Gambar Menu

## Lokasi Folder
Semua gambar menu harus disimpan di folder: `public/images/`

## Format Nama File
Nama file gambar harus sesuai dengan nama menu dengan format berikut:
- **Semua huruf KAPITAL**
- **Spasi diganti dengan underscore (_)**
- **Karakter khusus dihapus atau diganti**
- **Ekstensi: .jpg**

## Contoh Konversi Nama Menu ke Nama File

| Nama Menu | Nama File |
|-----------|-----------|
| Espresso | `ESPRESSO.jpg` |
| Caffee Latte | `CAFFEE_LATTE.jpg` |
| Long Black | `LONG_BLACK.jpg` |
| Coldbrew Original | `COLDBREW_ORIGINAL.jpg` |
| Coffee Frappe | `COFFEE_FRAPPE.jpg` |
| Gado-gado | `GADO_GADO.jpg` |
| Nasi Ayam Bakar Kasiko | `NASI_AYAM_BAKAR_KASIKO.jpg` |
| New York Cheese Cake Original | `NEW_YORK_CHEESE_CAKE_ORIGINAL.jpg` |

## Cara Menambahkan Gambar

1. Download gambar dari Google Drive atau sumber lain
2. Rename file sesuai format di atas (contoh: `ESPRESSO.jpg`)
3. Simpan file di folder `public/images/`
4. File akan otomatis terdeteksi oleh sistem

## Catatan Penting

- Pastikan semua file gambar memiliki ekstensi `.jpg`
- Nama file harus **PERSIS** sesuai dengan format di atas (case-sensitive untuk underscore)
- Jika gambar tidak ditemukan, akan ditampilkan placeholder dengan ikon kopi
- Ukuran gambar disarankan tidak terlalu besar (max 1-2MB per file) untuk performa yang optimal
